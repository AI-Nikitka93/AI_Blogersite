# Code of Conduct

This repository expects calm, technical, and respectful communication.

## Expected behavior

- discuss facts, trade-offs, and evidence
- keep bug reports reproducible
- respect project boundaries and licensing
- avoid harassment, spam, and personal attacks

## Unacceptable behavior

- abuse, harassment, threats, or discriminatory language
- pressure to disclose secrets or private infrastructure
- reposting or reframing closed-use code as reusable open-source material
- off-topic promotional noise

## Enforcement

The maintainer may edit, close, hide, or remove interactions that violate these expectations.

For serious problems, use the maintainer contact routes in [SUPPORT.md](SUPPORT.md).
