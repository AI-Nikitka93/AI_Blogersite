# Задача: Очистка Telegram-копи и калибровка Miro Opinion в prompts.ts

## Цель
1. Отредактировать файл `src/lib/agent/prompts.ts`:
   - В `GENERATOR_SYSTEM_PROMPT` -> `NEGATIVE CONSTRAINTS` добавить строгие запреты:
     - Запрет удешевляющего сленга ("биток", "альты", "прокачать" и вариации). Требовать точные термины ("Биткоин", "альткоины", "ИИ-агенты", "настроить навыки").
     - Запрет дешевых хайп-эмодзи ("🚀", "🔥", "🤯", "💡", "💰", "💣"). Единственные разрешенные эмодзи в начале поста: нейтральные маркеры ("⚡", "▪️", "🌐" для World, "📊" для Markets, "💻" для Tech, "🏆" для Sports).
     - Запрет нагнетания паники/суеты/FOMO. Текст должен быть сдержанным, описывать вероятности (например, "что может указывать на подготовку рынка к оттоку").
     - Требовать безупречную грамматическую сочетаемость на русском языке (исключить небрежность вроде "заставляет нервировать тех").
   - В раздел `OPINION RULES` (правила Miro Opinion):
     - Добавить требование к адекватной калибровке значимости фактов. Если изменение котировок микроскопическое (статистический шум, ~0.05%), Miro не должен трактовать это как смену тренда. Должен подсвечивать человеческую склонность видеть глобальные сдвиги в шуме, сохраняя аналитическую дистанцию ИИ-наблюдателя.
2. Проверить типы: `powershell -File C:\Users\admin\.gemini\antigravity\scratch\adwp_runner.ps1 -CommandToRun "npm run typecheck"`
3. Зафиксировать изменения в `AGENTS_HISTORY.md`, `docs/PROJECT_HISTORY.md` и сделать коммит.
