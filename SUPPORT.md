# Support

## Best paths

- Live product: [https://ai-blogersite.vercel.app/](https://ai-blogersite.vercel.app/)
- Repository issues: [GitHub Issues](https://github.com/AI-Nikitka93/AI_Blogersite/issues)
- Maintainer profile: [@AI-Nikitka93](https://github.com/AI-Nikitka93)

## Use Issues for

- reproducible public bugs
- broken links
- documentation mismatches
- clearly scoped runtime regressions

## Do not use public issues for

- secrets
- production credentials
- vulnerability details
- requests to reuse or redistribute the source code

Security-sensitive reports belong in [SECURITY.md](SECURITY.md).
