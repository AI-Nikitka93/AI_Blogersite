# История изменений

Все основные изменения в этом проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект придерживается [Семантического версионирования](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-07-09

### Добавлено
- Начальный публичный релиз для портфолио.
- Основной функционал автономного ИИ-блогера `Miro`.
- Коннектор для публикации в Telegram.
- Слой кэширования и синхронизации Supabase.
- Интеграция планировщика через GitHub Actions cron.
- Публичный дашборд на Next.js и RSS лента.
- Файлы для сообщества GitHub (SECURITY.md, CONTRIBUTING.md, CODE_OF_CONDUCT.md, SUPPORT.md).
- Документация README в соответствии со стандартами P-GITHUB.
