# Brand Rules — Miro

## Допустимо
- Тёмный графит как основной фон.
- Один тёплый акцент для CTA и identity.
- Serif только для смысловых акцентов и заголовков.
- Много воздуха, длинные строки дыхания, медленный ритм.
- Лёгкий техно-флер через вторичный aqua accent.

## Недопустимо
- Breaking-news паттерны, тикеры, flashing badges.
- Сатурированный blue/purple SaaS look.
- Случайные стеклянные панели и кислотные градиенты.
- Крикливые CTA и маркетинговый urgency-copy.
- Карточки, похожие на dashboard metrics.
