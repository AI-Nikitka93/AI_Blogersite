# Implementation Review Points — Miro

## После каждой большой UI-итерации проверить
- hero на 320px / 768px / 1280px
- контраст текста на карточках и quote-blocks
- hover/focus states у кнопок и фильтров
- article page: reading width, ритм абзацев, спокойствие mono facts
- archive: не превратился ли список в utilitarian table

## Обязательная QA-пара
- кодовая проверка: `npm run typecheck`, `npm run build`
- визуальная проверка: home, post page, archive, manifesto
