# Deprecation Rules — Miro

- Старый компонент нельзя удалять, пока новый не покрывает те же состояния.
- Депрекейт оформляется в 2 шага:
  1. маркируем usage stop в docs;
  2. удаляем после замены на всех страницах.

## В первую очередь нельзя ломать
- serif headline system
- mono Observed blocks
- diary-thread signature
- category navigation rhythm
